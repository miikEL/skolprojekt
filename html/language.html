HTML:

HTML was used to create the elements on the DOM object.
These elements include but are not limited to: BODY, HEAD, DIV, A, P, H1.

CSS (BOOTSTRAP):

CSS mutated the elements on the DOM, the mutation could be a change in colors, size or positioning.
Bootstrap was mainly used to position the objects on the page, it also makes the page responsive
meaning that it changes size depending on screen size to fit the screen.

Javascript (jQuery):

JAVASCRIPT added functionality to the elements on the DOM.
jQuery is a JavaScript library meaning that jQuery is essentially JavaScript code
that has been used to create methods/functions for other developers to use.
The jQuery makes the code more available to users as it's easier to use,
it's also been tested by millions of users meaning that the code is good efficient code.
